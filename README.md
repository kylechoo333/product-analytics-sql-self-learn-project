# product-analytics-sql-self-learn-project
SQL-based product analytics project analyzing user behavior, retention, and revenue (DAU, day-1 retention, channel performance)

## Key Analyses
- Signup channel performance
- Daily Active Users (DAU)
- Revenue by acquisition channel
- Day 1 retention
- Early engagement vs revenue

## Tables
- users
- events
- transactions

## Skills Demonstrated
- SQL joins
- Aggregations (COUNT, SUM)
- CTEs
- CASE WHEN logic
- Retention analysis

## How to Run
1. Run schema.sql
2. Run sample_data.sql
3. Run analysis.sql
